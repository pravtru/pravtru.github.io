Блоки используются для объединения повторяющихся элементов.
Блоки содержат изменяемые параметры, которые передаются в вызывающем теге. Далее вызванные параметры используются для подстановки в элементы блока и изменения графического представления. Для каждого тега в блоке имеющего id будет произведена замена свойства и значений на соответствующие свойства и значения параметра.
